Homeless Analysis & Solutions

Project Description -

  • Designed a web application (REST Web Services) to provide statistical analysis of homeless people based on different factors such as gender,housing,veterans,reason etc.
  • Also provided best possible solutions and services to end homelessness based on our analysis
  • Used Google chart APIs to display statistical analysis.
  • Technologies: Javascript, HTML5, CSS, BootStrap,NodeJS, MongoDB and Redis
