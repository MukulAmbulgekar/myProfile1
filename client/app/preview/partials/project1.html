eBay - Online Market Place

Project Description -

  • Designed a 3-tier web application (REST APIs) with basic functionalities of online market system exactly similar to eBay.
  • Functionality - Log in session handling,Buyer/Seller, Auction/Bidding, CRUD operations on users and products.
  • Performance Tuning Techniques - Connection Pooling, SQL and Object Caching (Redis Cache), Data Denormalisation, Prepared statements.
  • Performed Server-Side and Client-Side validations for duplicate user,strong password,zip code pattern.
  • Technologies: Javascript, HTML5, CSS, BootStrap, AJAX, JQuery, NodeJS, MySQL, Redis, JMeter and Mocha.
